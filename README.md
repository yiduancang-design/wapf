# 文案评分优化 · WAPF

> 足球短视频口播文案的 AI 评分、二次优化、单场深度拆解与下粉话术生成 Skill 备份库。
>
> 本仓库用于保存完整的 `wenan` 技能文件，方便在其他电脑下载、安装，也方便后续版本继续同步到 GitHub。

---

## 这个项目能做什么

`wenan` 是一个面向足球短视频口播内容的文案评分优化工作流。它不是简单润色，而是按“足球判断力 + 短视频传播力 + 口播表达 + 转化安全”四条线一起检查。

核心能力：

- **文案评分**：对足球口播文案进行快速模式或完整模式评分。
- **二次优化**：在保留原观点和表达风格的基础上，强化开头、逻辑、节奏、可信度与完播率。
- **二次复评**：优化稿输出后继续打分，确认最终稿是否达到 8 分以上。
- **单场深度拆解**：围绕一场比赛形成“结论、证据、战术翻译、风险、方向”的闭环。
- **下粉话术**：生成关注、收藏、评论、直播预约等平台内转化话术，同时规避站外导流和收益承诺风险。

## 两种使用方式

### A. 在当前电脑使用

在 Codex 中直接输入：

```text
$wenan
```

然后粘贴你的足球短视频口播文案，或说明你要做的任务，例如：

```text
$wenan 帮我给这条足球口播文案评分并二次优化
```

```text
$wenan 帮我做一场曼城 vs 皇马的单场深度拆解，并生成下粉话术
```

首次触发时，Skill 会让你选择：

- **快速模式**：适合日常高频改稿，输出更短、更快。
- **完整模式**：适合重要作品，包含完整四维评分、问题拆解、优化稿与复评。

### B. 在另一台电脑安装

1. 打开仓库：`https://github.com/yiduancang-design/wapf`
2. 点击 **Code → Download ZIP**，或使用 Git 克隆：

```bash
git clone https://github.com/yiduancang-design/wapf.git
```

3. 把仓库里的 `skill/` 文件夹复制到 Codex skills 目录，并重命名为 `wenan`。

Windows 常见路径：

```text
C:\Users\你的用户名\.codex\skills\wenan
```

最终结构应类似：

```text
C:\Users\你的用户名\.codex\skills\wenan\SKILL.md
C:\Users\你的用户名\.codex\skills\wenan\references\...
C:\Users\你的用户名\.codex\skills\wenan\agents\openai.yaml
```

4. 重启 Codex 或开启新对话后，输入 `$wenan` 即可使用。

## 推荐工作流程

### 1. 输入原稿

你可以直接粘贴口播文案，也可以只给出比赛、观点、方向，让 Skill 帮你生成初稿。

建议输入包含：

- 比赛双方
- 你的核心判断
- 想做的账号风格
- 是否需要引导关注、收藏、评论或直播预约
- 是否偏向激进观点、稳健观点或数据型表达

### 2. 初次评分

Skill 会从这些角度检查：

- 开头是否有钩子
- 观点是否清楚
- 足球逻辑是否自洽
- 证据是否能支撑判断
- 口播节奏是否顺
- 是否有模板感、废话、空话
- 是否存在违规、夸大、收益承诺或站外导流风险

### 3. 二次优化

优化时会尽量保留你的原观点和口语风格，只改真正影响传播的地方：

- 开头更抓人
- 逻辑更顺
- 信息密度更高
- 句子更适合口播
- 结尾更能引导互动
- 风险表达更安全

### 4. 优化后复评

优化稿输出后会再次评分，重点看：

- 是否达到 8 分以上
- 原来的扣分点是否已经修复
- 新稿是否更像能发布的成片口播
- 是否还需要继续追问或小改

### 5. 单场深度拆解

当你要围绕一场比赛做深度内容时，可以要求：

```text
$wenan 帮我做这场比赛的单场深度拆解
```

它会按以下链路组织：

```text
结论 → 证据 → 战术翻译 → 风险 → 最终方向
```

适合做赛前预测、赛后复盘、直播预热视频和观点型账号内容。

### 6. 下粉话术

可以生成平台内安全转化话术，例如：

- 关注型
- 收藏型
- 评论型
- 直播预约型
- 系列内容预告型

会自动避开：

- 收益承诺
- 暗示稳赚
- 站外导流
- 违规暗语
- 过度刺激和诱导表达

## 后续如何同步版本

本仓库建议把 `skill/` 作为唯一技能源。

后续你在本地更新 Skill 后，只需要把这些文件同步到 GitHub：

```text
skill/SKILL.md
skill/agents/openai.yaml
skill/references/*.md
```

在另一台电脑更新时，进入仓库目录后执行：

```bash
git pull
```

或者重新下载 ZIP，再覆盖本地 `wenan` 技能文件夹即可。

## 注意事项

- 本 Skill 用于文案质量评估和表达优化，不承诺比赛结果准确。
- 涉及赛果、盘口、概率、收益等内容时，应保持克制表达。
- 下粉话术只建议做平台内关注、收藏、评论、直播预约，不建议做站外导流。
- `README.md` 只负责说明；真正的技能规则请看 `skill/SKILL.md`。
