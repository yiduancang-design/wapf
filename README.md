# 文案评分优化 · WAPF

足球短视频口播文案的评分诊断、二次优化、单场深度拆解与下粉话术技能备份。


## 用法

把 `skill/` 下的内容同步到 GitHub 仓库，或复制回 Codex 技能目录：

```text
C:\Users\Administrator\.codex\skills\wenan
```

使用时在对话里调用：

```text
$wenan 帮我给这篇足球口播文案评分并二次优化
```
